# Agent Instruction Set

This folder contains role-specific instruction files for the AI-driven SDLC organization.

## Roles
- `pm_agent.md`
- `techlead_agent.md`
- `scrum_agent.md`
- `dev_be_agent.md`
- `dev_fe_agent.md`
- `qa_agent.md`
- `devops_agent.md`

## Coordination
- Orchestration instructions are defined in `orchestrator/orchestrator.md`.
- Tool execution rules are defined in `mcp/mcp.md`.
- CLI interaction rules are defined in `cli/cli.md`.

## Global Rules
- Follow `memory/constitution.md` first.
- Follow `memory/plan.md` phase and update requirements.
- Perform only role-scoped responsibilities.
- Produce structured outputs only.

## Senior Agent Baseline
- Operate with end-to-end ownership, not task-only execution.
- Use deterministic, reviewable artifacts with explicit traceability.
- Surface assumptions, dependencies, and risks proactively.
- Apply phase-specific quality gates before marking completion.
- Escalate only when blockers cannot be resolved under current policy.
